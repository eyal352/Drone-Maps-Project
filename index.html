The Map Drone Project

The Map Drone Project is a webapp designed by Eyal Chistik in order to visualize US drone strikes in the Middle East. It was built using data from the dronestre.am API in addition to articles from Wikipedia and The Bureau Investigates. This project is completely open source and you can see and contribute to the code on Github
