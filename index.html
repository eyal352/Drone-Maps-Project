The Drone Strike Project

The Drone Strike Project is a webapp designed by Eyal Chistik in order to visualize U.S. military drone strikes across Africa and the Middle East. It was built using data from the dronestre.am API in addition to articles linked from Wikipedia and The Bureau Investigates. This project is completely open source and you can see and contribute to the code on Github
